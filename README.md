# Features:

* **Core Tech & UI:**
    * direct P2P.
    * Room Codes.
    * Animated themes.
    * Fast on phone or PC.

* **Character Management:**
    * Easy builder.
    * Auto-math for all stats.
    * Track health, spells, & Inventory.
    * Custom crest maker (for P2P).
    * Export/Import characters.

* **Dungeon Master Tools:**
    * Dashboard for organized session.
    * Fast dice & math counters.
    * NPC & Beasts creation.
    * Categorized Notes.
    * calendar.
    * Random generators.
    * Full database backups.

* **Combat & World Map:**
    * Move tokens on a map.
    * Turn-order tracker.
    * Draw walls & map shapes.
    * Animated weather effects.

* **Session Interaction:**
    * Shared themes for everyone.
    * Live shared dice rolls.
    * Screen-shake for big hits.
    * Chat as your NPC characters.

* **Library & Homebrew:**
    * Search all spells & rules.
    * Click text to roll dice.
    * Create custom races & classes.
